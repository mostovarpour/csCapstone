Project Details

Description

  • A gigapixel image is composed of one billion pixels.
  • That is 1000 times the image data captured by a 1 megapixel camera.
  • For this project we are going to develop a way to quickly and accurately view
  • very large images without losing clarity or taking a very long time.

Requirements

  • Improve on the performance and flexibility of the sampling process.
  • Develop a method to quickly view images while enhancing quality in the background.
  • Create a product that is able to interact with GDAL.
  • Develop a cross-platform viewer that can quickly view images but still retains functionality.

High-Level Design

  • Our main design goal is to create an image viewer (most likely using C) that will
  • be able to view very large images in a very short amount of time.
  • If we are able to develop an efficient reader then we will also develop our own image viewer.

Tools

  • GDAL
  • TuiView
  • BGFX
  • C/C++/Python/OpenGL
